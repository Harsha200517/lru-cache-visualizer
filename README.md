# LRU Cache Visualizer

A simple web-based project to demonstrate how the LRU (Least Recently Used) Cache works using a visual interface.

---

## 🔍 What is LRU Cache?

LRU Cache is a data structure used to store limited data.  
When the cache is full, it removes the least recently used item.

---

## 🚀 Features

- Add data to cache
- Access existing data
- Automatic removal of least used data
- Visual representation (Top = Most Recent, Bottom = Least Recent)
- Cache hit and miss tracking
- Activity log of operations
- Clean and simple dark UI

## 📸 How It Works

- Add key → stored in cache
- Access key → moves to top (most recently used)
- When full → bottom element (least used) is removed
